Big Picture

한국 공구의 진짜 구조

핵심 발견: 한국 공구제안서는 슬라이드 덱이 아니다. 1-페이지 Excel 표다. 상품 행 + 마진 컬럼 + 운영 조건 = 끝. 브랜드 스토리·시장 분석·페르소나 — 모두 다른 자리에서 다룬다. 공구제안서는 "숫자가 맞나" 하나의 질문에 답하는 문서다.

샘플규정
"공동구매 진행시 1박스 무상 지원 (공구 미진행시 정산에 반영)" — 셀러가 공구 진행 약속 시 샘플 무상, 진행 안 하면 샘플비 정산에서 차감.
배송비
택배사 (CJ대한통운 / 한진택배) · 기본 3,500원 / 제주·도서산간 +4,000원 / 왕복 반품 7,000~8,000원. 1박스 = 고객부담 / 2박스+ = 셀러부담 (무료배송). 셀러가 자기 마진에서 배송비를 흡수하는 구조.
교환/반품 기준
1. 제품 하자 또는 구매자 귀책사유 아닌 파손 → 무상 교환/반품 가능 / 2. 단순변심 → 왕복 택배비 구매자 부담
배송정책
"당일 12시 이전 구매 → 당일 배송 / 당일 12시 이후 구매자 → 익일 발송" — 12시 cutoff이 한국 공구 표준. 일부는 11시.
발주마감
12시 (or 11시) — 매일 cutoff. 이 시간 전 주문은 당일 발송.
발주양식
"브랜드 자체 양식 전달" — 각 브랜드가 Excel 양식 가지고 있고 셀러한테 전달. 양식엔 SKU/수량/배송지가 들어감.
정산
"공동구매 마감 후 7–14영업일 이내 정산 / 세금계산서 발행". 자사몰 기준은 7영업일, 셀러 링크 기준은 3영업일이 더 흔함. D+7 ~ D+14이 표준 범위.
VAT 포함
F&B 카테고리에서는 모든 가격을 (VAT 포함) 으로 명시. 화장품도 비슷.

전략 권장사항

AVO 공구 운영 룰

Rule 01 · 벤더 없이 시작

초기엔 인플루언서와 DIRECT 거래. 셀러마진 35% + 벤더 0%. 벤더 끼우면 추가 10% 빠짐 → 브랜드 마진 절반으로 줄어듬. 충분히 자체 매칭 가능한 인플루언서 풀 (Reviews v3 보낼 친구·인플루언서 35명) 이 있으니 굳이 벤더 필요 없음.

Rule 02 · 정상가 = Naver/Coupang 그대로 (39,900)

정상가 = 현재 Naver/Coupang 가격 그대로 (39,900원). 공구판매가 29,900원 = 25% 할인. 공구 참여자만 누리는 한정 특가. 인플루언서·관계 기반 구매라 가격 차별 acceptable. 단, Naver/Coupang 정가는 그대로 유지 — 인하 X.

Rule 03 · 4-tier 구성 (15/30/45/60팩)

1박스 (15팩) / 2박스 (30팩) / 3박스 (45팩) / 4박스 (60팩). 15팩 박스 고정 → 자연스럽게 4-tier로 정착. 사은품 escalation: 없음 / +1팩 / +3팩 / +5팩+텀블러.

Rule 04 · 셀러마진율 20-25% 2-옵션

옵션 A 25% (Standard, 첫 협업) · 옵션 B 20% (우대 단가, 약정 협업). 박스 구성 무관하게 옵션 내 마진율 일정 유지. F&B 표준 35% 대비 낮은 단가지만, AVO의 콘텐츠·키워드·CS 지원이 셀러 작업 부담 줄여줘서 협상 가능한 단가.

Rule 05 · 정산 = D+7

공구 마감 후 7영업일 정산. 셀러 입장에서 빠른 정산이 reorder 결정에 가장 큰 영향. 14일 정산보다 7일 정산이 차별화 포인트.

Rule 06 · 배송비 정책 명확

1박스 = 고객부담 3,500원 / 2박스+ = 셀러부담 무료. 셀러가 자기 마진에서 배송비 흡수해서 소비자한테 "무료배송" 어필. 표준 룰이라 셀러도 받아들임.

Rule 07 · 발주마감 12시

당일 12시 cutoff = 당일 발송 / 12시 이후 = 익일 발송. 표준 룰. 박정은 대표 (광장동 창고) 가 매일 12시 cutoff 운영 가능한지 확인 필요.

Rule 08 · 자사몰 무, 셀러 링크 중심

AVO는 현재 자사몰 없음. Naver/Coupang 셀러링크 방식 → 셀러가 본인 인스타·블로그에 링크 걸고 매출 추적. 정산은 마감 후 3영업일.

Anti-pattern · 하지 말 것

1. 공구판매가 너무 낮추지 말 것 — 가격 정책 깨짐. 현재 공구가 29,900원 vs Naver/Coupang 39,900원 = 25% 차이. 공구 한정 특가라 acceptable하지만, 25,000원 이하로 내려가면 Naver/Coupang 일반 구매자 신뢰 무너짐. 또한 Naver/Coupang 정가는 절대 인하 X.

2. 셀러마진율 18% 미만으로 떨어뜨리지 말 것. AVO 기준은 20-25% (옵션 A/B). 그 이하는 relationship-driven pool이라도 의욕 떨어짐. 약정 협업 (옵션 B) 도 20% floor 유지. 첫 협업 (옵션 A) 25% 절대 양보 X.

3. 정산 14일 이상으로 연장하지 말 것. 셀러 자금회전 영향 큼. 7일 정산이 차별화 포인트가 되는 시장이라 14일도 표준이지만 그 이상은 reorder 안 일어남.

4. 무리한 사은품 약속 안 됨. 4박스 = +1박스 (9,000원 비용) 던지면 브랜드 마진 압박. 우리는 +5팩 + 텀블러 조합 (총 비용 6,000원, perceived value ~16,300원) 으로 효율적 incentive 구성.

5. "효과·효능" 언어 셀러 카피에 들어가지 않게 사전 안내 — 일반식품 라인. 셀러가 본인 카피에 "면역력 강화" 같은 표현 쓰면 식약처 단속 → 우리 책임. 공구제안서 비고란에 "본 제품은 일반식품. 효능 표현 금지" 명시 필요.

2030 인플루언서 풀 매칭

Reviews v3 보낼 친구·가족·인플루언서 35명 중 공구 가능한 후보를 1~3명 선정해 첫 공구 진행. 첫 공구 = pilot (목표 GMV 5~10백만 원, brand net 1~2백만 원). 결과 보고 2차 공구는 더 큰 인플루언서 (1만~5만 팔로워) 로 확장.
